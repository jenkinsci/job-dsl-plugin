System.setSecurityManager(..)

Uses System.setSecurityManager(..); to set org.codehaus.groovy.tools.shell.util.NoExitSecurityManager() as the new SecurityManagerfor the whole Jenkins Master JVM.
This will prevent a System.exit(); call in JobDSL GroovyScripts to prevent a Jenkins shutdown.

If the box is unchecked, the System SecurityManagager will be set to "null".
This will only happen if the previous SecurityManager was an instance of org.codehaus.groovy.tools.shell.util.NoExitSecurityManager.
